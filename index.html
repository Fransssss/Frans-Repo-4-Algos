Algorithm Portfolio

This collection highlights multiple algorithm paradigms, showcasing different problem-solving techniques and their implementations.

Problem 1 — Greedy Algorithm

Description: The Interval Scheduling Maximization problem asks you to select the largest possible subset of intervals (each with a start and end time) such that no two chosen intervals overlap, e.g., scheduling as many non-conflicting events as possible in one lecture hall.

Repository Link: https://github.com/Fransssss/interval-scheduling-greedy.git

Problem 2 — Divide and Conquer

Description: This lightweight CLI demo animates how merge sort applies Divide and Conquer by splitting a sample list, solving each half, and merging the results so you can watch the full recursive process unfold in real time.

Repository Link: https://github.com/Fransssss/merge-sort-divide-and-conquer.git

Problem 3 — Dynamic Programming

Description: This app illustrates the Longest Common Subsequence dynamic-programming algorithm by letting you enter two strings, optionally watch the DP table fill in real time, and then see the computed table, LCS length, and one matching subsequence.

Repository Link: https://github.com/Fransssss/longest-common-subsequence-dynamic-programming.git

Problem 4 — NP / Heuristic Algorithm

Description: This small Python demo builds a sample graph, explains the NP-complete vertex cover problem, and showcases a greedy 2-approximation whose step-by-step execution is animated in the terminal for easy visualization.

Repository Link: https://github.com/Fransssss/vertex-cover-heuristic.git

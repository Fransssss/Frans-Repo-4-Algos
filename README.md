# Algorithms Portfolio

## Overview
This repository hosts a GitHub Pages portfolio for a computer science algorithms course. The site highlights how different algorithm paradigms are implemented across multiple dedicated repositories, making it easy to compare techniques and study their problem-solving strategies.

## Portfolio Structure
The portfolio surface links to four core paradigms:
- Greedy Algorithm
- Divide and Conquer
- Dynamic Programming
- NP / Heuristic Algorithm

Each card in the site includes a short description and a placeholder spot where you can plug in the GitHub repository demonstrating that paradigm.

## Notes
Each linked repository is expected to include:
- An algorithm explanation
- Source code implementing the technique
- Example execution or demo instructions
- Complexity analysis discussing time and space costs

Use this structure to keep the overall portfolio consistent and informative for course reviewers or potential collaborators.
